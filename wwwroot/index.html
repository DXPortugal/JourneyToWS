The Program

Journey to the Web Summit is a Microsoft Program designed to support startups in making the most of Websummit as well as showcase the top European Startups. The Program comprises two initiatives:

Startup Challenge

Challenge to select the top European startups to be showcased at the Microsoft booth @ Web Summit

Online Masterclasses

Business and Technical sessions to support startups in getting ready for and making the most out of Websummit
